# Contributing to MeltingICE

Thanks for your interest in contributing. This project is safety-focused, so please be thoughtful and careful.

## Getting started

1. Fork the repo and create a branch from `master`.
2. Install dependencies and run the app locally.
3. Keep changes focused and small when possible.

## Development setup

- Frontend: `npm install` then `npm run dev`
- Backend: configure `backend/.env` and run PHP locally

## Pull requests

- Describe the problem and your solution.
- Include screenshots or logs for UI or behavior changes.
- Add tests if you change logic.

## Code style

- Follow existing patterns in the repo.
- Prefer readable, explicit code over cleverness.
- Avoid committing secrets or data exports.

## Reporting safety issues

Please see `SECURITY.md`.
